## Responsive Breakpoints

### Desktop (> 1200px)
- Full table width
- All columns visible
- Horizontal severity filters
- Side-by-side search and category

### Tablet (768px - 1200px)
- Horizontal scroll for table
- All columns visible
- Horizontal severity filters
- Stacked search and category

### Mobile (< 768px)
- Horizontal scroll for table
- Smaller severity filter buttons
- Stacked filters
- Reduced padding
- Smaller font sizes

## Accessibility Features

### ARIA Labels
- Severity filter buttons: "Filter by [severity] severity, [count] alerts"
- Search input: "Search alerts by message, type, or action"
- Category dropdown: "Filter alerts by category"
- Sort headers: "Sort by [column name]"
- Pagination: "Select page of alerts"

### Keyboard Navigation
- Tab: Navigate through interactive elements
- Enter/Space: Activate buttons and filters
- Arrow keys: Navigate table cells
- Escape: Clear active filters

### Screen Reader Announcements
- Filter changes: "Showing [count] alerts filtered by [criteria]"
- Sort changes: "Sorted by [column] in [direction] order"
- Page changes: "Showing page [number] of [total]"

## Performance Metrics

### Target Load Times
- Initial render: < 100ms
- Filter application: < 50ms
- Sort operation: < 50ms
- Page change: < 50ms
- Search (after debounce): < 100ms

### Data Handling
- Max alerts per page: 10 (default), 25, 50
- Total alerts supported: 1000+ (with pagination)
- Search index: All text fields
- Filter combinations: Unlimited
